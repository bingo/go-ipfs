kickDevice method

void kickDevice ({@required String userName, @required String password, @required String resource, dynamic onError(int code, String desc) })

kickDevice - kick device out. Access control by userName/password pair, device identified by resource. If anything wrong, onError will be called.

Implementation

void kickDevice(
    {@required String userName,
    @required String password,
    @required String resource,
    onError(int code, String desc)}) {
  Future<Map> result = _emClientChannel.invokeMethod(EMSDKMethod.kickDevice,
      {"userName": userName, "password": password, "resource": resource});
  result.then((response) {
    if (!response['success']) {
      if (onError != null) onError(response['code'], response['desc']);
      return null;
    }
  });
}
